**Проверка гипотез о зависимости (корреляции) между данными**

*********************************************************************************************************************************************

1. Зависимость между количеством товаров в заказе и размером скидки к заказу.
   
   H0: данные независимы, H1: данные зависимы.

   Данные числовые, распределены ненормально, выбросы есть - используем корреляцию Кендалла.

   Оцениваем только невозвратные заказы.

   pvalue=0.79 > 0.05 - (корреляция =-0.002) гипотезу о независимости  количества товаров и скидки не отвергаем.

*********************************************************************************************************************************************

2. Зависимость между между типом доставки  и прибылью.
   
   H0: данные независимы, H1: данные зависимы.

   Данные порядковые, распределены ненормально, выбросы есть - используем корреляцию Спирмена.
   
   Оцениваем только невозвратные заказы.

   pvalue=0.19 > 0.05 - (корреляция =-0.01) гипотезу о независимости  типа доставки и прибыли не отвергаем.

*********************************************************************************************************************************************

3. Зависимость между между типом доставки  и  количеством товаров в заказе.
   
   H0: данные независимы, H1: данные зависимы.

   Данные порядковые, распределены ненормально, выбросы есть - используем корреляцию Спирмена.
   
   Оцениваем все заказы.

   pvalue=0.076 > 0.05 - (корреляция =0.018) гипотезу о независимости  типа доставки и количеством товаров в заказе не отвергаем.

*********************************************************************************************************************************************

4. Зависимость между между суммой заказа и фактом возврата. 
   
   H0: распределения равны (медианы), H1: распределения не равны.

   Данные бинарные и числовые, распределены ненормально - используем непараметрический H-критерий Краскела-Уоллиса проверяем гипотезу о равенстве распределений.
   
   Оцениваем случайные выборки одинакового размера невозвращенных и возвращенных товаров.

   pvalue=0 < 0.05 - гипотезу о равенстве дисперсий сумм заказа при невозвратных и возвратных заказах отвергаем.

   Рассчитываем кореляцию для бинарных и числовых данных.

   corr = -0.085 - гипотезу о зависимости между суммой заказа и фактом возврата отвергаем, очень слабая отрицательная связь.

*********************************************************************************************************************************************

   H0: данные независимы, H1: данные зависимы.
