Если хотя бы один заказ был совершен, то это успех, если нет, то неудача. Оценим, одинаковый ли эффект дали эти акции? Если разный, то между какими акциями наблюдается различие?

   H0: все 3 акции дали одинаковый эффект,  H1: все 3 акции дали разный эффект.
