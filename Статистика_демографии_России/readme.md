1.	Гипотеза о равенстве среднего уровня рождаемости в России в 2015, 2016 и 2017 году. Предполагается, что данные имеют нормальное распределение.

Н0: дисперсии равны. Н1: дисперсии не равны.
 
p-value 0,719 > 0.05 – не отвергаем гипотезу о равенстве дисперсий. Значит, мы можем воспользоваться односторонним дисперсионным анализом.

Н0: средние равны. Н1: средние не равны.
 
pvalue много меньше заданного уровня значимости 0.05. Значит гипотезу о равенстве средних отклоняем.

Воспользуемся тестом Тьюки для того, чтобы определить, какая именно группа отличается.
 
Попарное сравнение гипотез показало, что лишь гипотезу о равенстве средних за 2015 и 2016 год нельзя отклонить (pvalue = 0.409 > 0.05). Все остальные гипотезы о равенстве нужно отклонить.

Н0: дисперсии равны. Н1: дисперсии не равны.

Н0: средние равны. Н1: средние не равны.
 
p-value — это экстремально маленькое значение (практически ноль). Отвергаем нулевую гипотезу о равенстве средних между этими регионами. 

Средние значения показателя urbanization статистически значимо различаются между регионами.

Проверим, какие группы отличаются, с помощью теста Тьюки.

Н0: средние равны. Н1: средние не равны.
 
На уровне значимости 0,05 не можем опровергнуть гипотезу о равенстве средних только для Алтайского края и Астраханской области.
