1. Определени количества уникальных пользователей, которое участвовало в эксперименте.

2. Определение контрольных групп. 

3. Определени в датасете  пользователей, которые попали в обе группы, их количество.

4. Проверка метрик:

   Проверка разницы в количестве пользователей (0.7% < 1%) - условие соблюдается.

   Проверка ARPU - по группам соответствует. 

   Проверка среднего количества покупок - одинаковое.
   
   Проверка среднего чека - одинаковые.

5. Проверка гипотез об отличии групп в А/А-тесте.

   p_value=0.407 > 0.05 - гипотезу о равенстве распределений выручки в группах A1 и A2 не отвергаем с уровнем значимости a=0.05.

   p_value=0.23 > 0.05 - гипотезу о равенстве распределений количества покупок в группах A1 и A2 не отвергаем с уровнем значимости a=0.05.

   p_value=0.145 > 0.05 - гипотезу о равенстве распределений суммарного дохода от пользователя в группах A1 и A2 не отвергаем с уровнем значимости a=0.05.
