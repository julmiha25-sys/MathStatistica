**Выбор данных:**

**Задачи:**

1.	Анализ деятельности менеджеров в разрезе филиалов (насколько эффективно каждый мене-джер обслуживает каждый филиал по количеству заказов) с целью выявления филиалов с де-фицитом заказов.
   
2.  Анализ клиентов по объему заказов с целью выявления общей тенденции работы компании – ориентация на корпоративных клиентов или нет.
   
3.  Проверка прогноза количества продаж по направлениям бизнеса с целью проверки ожиданий.
   
4.  Анализ эффективности скидочных акций с целью управления акционной политикой.
   
5.  Анализ разнообразия заказов с целью выявления приоритетного направления бизнеса, вида услуг, филиала.
    
6.	Анализ прибыли компании за 2 периода для определения успешности бизнеса в долгосрочной перспективе.
    
7.	Анализ зависимости объема перевозок от вида услуг с целью выявления приоритетных видов услуг.

H0: распределение количества заказов по клиентам за 2016 год является нормальным.

H1: распределение количества заказов по клиентам за 2016 год не является нормальным.

По критерию К-квадрат Д'Агостино: для обоих выборок p_value =0 < a=0.05 - гипотезу о нормально-сти выборки отвергаем на уровне значимости α = 0.05.

7.	Анализ распределения объема перевозок  для всех видов услуг и всех филиалов в отдельности.

H0: распределения объема перевозок одинаковы по всем видам услуг.

H1: распределения объема перевозок не одинаковы по всем видам услуг.

Визуализируем данные:
 
На боксплоте видно, что выбросы есть,  скорей всего данные имеют не нормальное распределение.

По критерию Kruskal-Wallis (не требует нормальности распределения и устойчив к выбросам) p_value=0.07 > 0.05 – гипотезу об одинаковом распределении объема перевозок по всем видам услуг не отвергаем на уровне значимости α = 0.05.

Вывод: нет статистически значимых различий в объемах перевозок между разными видами услуг.

H0: распределения объема перевозок одинаковы по всем филиалов.

H1: распределения объема перевозок не одинаковы по всем филиалам.

Визуализируем данные:
 
На боксплоте видно, что выбросы есть,  скорей всего данные имеют не нормальное распределение.

По критерию Kruskal-Wallis (не требует нормальности распределения и устойчив к выбросам) p_value=0.007 < 0.05 – гипотезу об одинаковом распределении объема перевозок по всем филиалам отвергаем на уровне значимости α = 0.05.

Выявим филиалы, статистически значимо отличающиеся по объёму перевозок.

По Dunn's test: ни одна пара филиал-объем перевозок не показала p_value < 0.05.

**Выводы:**

•	Kruskal-Wallis тест выявил статистически значимые различия между филиалами по объемам перевозок (p_value = 0.007). 

•	Однако попарные сравнения (Dunn's test с поправкой Бонферрони) не обнаружили значимых различий филиалов по объемам перевозок на уровне значимости α = 0.05. 

•	Наиболее близки к значимости различия между Москвой и Тюменью (p = 0.067): по бокспло-ту видно, что медиана Москвы больше, чем у Тюмени. 

•	Отсутствие значимых парных различий при общем значимом результате Kruskal-Wallis может быть связано с: небольшим размером выборки (n = 50); высокой консервативностью поправки Бонферрони; различиями, распределенными между несколькими группами.
