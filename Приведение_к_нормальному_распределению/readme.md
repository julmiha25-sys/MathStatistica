**Приведение к нормальному распределению**

Анализ распределения зарплат в статистических данных по России.

1. Какое преобразование сделало данные скошенными влево? Логарифмическое, преобразование квадратного и кубического корня.

2. Какое преобразование больше всего отклонено от симметричного распределения? Преобразование обратная дробь.

3. Какие данные оказались лучше преобразованных натуральным логарифмом? Исходные, преобразование квадратного и кубического корня.

4. Какие данные хуже всего согласуются с нормальным распределением? Преобразование обратная дробь.
