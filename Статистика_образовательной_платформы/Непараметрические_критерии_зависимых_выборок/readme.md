**Проект по непараметрическим критерям зависимых выборок**

На образовательной платформу произошло два серьезных изменения с задачами по программированию. 

В первом обновлении добавили новый метод проверки кода, во втором - добавили фидбек по решению и изменили дизайн платформы. 

Есть случайная выборка 30 клиентов. До релизов и после каждого релиза они прошли один и тот же опрос, состоящий из трех вопросов. 

1. Первый вопрос: "Довольны ли вы качеством проверки кода?"

2. Второй вопрос: "Достаточно ли быстро проверяется код?"

3. Третий вопрос: "Оцените от 1 до 10 дизайн платформы"

Нужно проанализировать, повлияли ли изменения платформы на реакцию пользователей и, если можно, оценить повлияло в лучшую сторону или в худшую.

Н1: новый метод проверки кода повлиял на реацию пользователей.

Гипотеза о нормальности 1 выборки отвергнута по одновыборочному критерию Колмогорова-Смирнова.

Непараметрический критерий Фридмана для 3-х выборок показал: p_value=0.029 < a=0.05.

Отвергаем гипотезу о том, что выборки взяты из одного распределения - новый метод проверки кода оказал влияние на пользователей.

Определим какая выборка оказала существенное влияние.

Используем критерий Вилкоксона для 2-х попарных выборок (при множественных сравнениях выбрается свой уровень значимости - 0.017, для 3-х групп).

В результате все значения p_value > a=0.017 - гипотезу о том, что выборки взяты из одного распределения и изменение кода не оказало влияние на пользователей не отвергаем.

Вывод: Несмотря на то, что критерий Фридмана выявил статистически значимые различия между тремя опросами, попарное сравнение с поправкой не подтвердило значимых изменений ни между одной парой опросов (p>0.017).
Это может указывать на разнонаправленные индивидуальные изменения оценок, которые не привели к систематическому сдвигу. 

Рекомендация: провести тест повторно для большей выборки.
